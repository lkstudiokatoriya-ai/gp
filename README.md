🎓 Polytechnic Hub

Learn • Practice • Succeed

Polytechnic Hub is a Progressive Web App (PWA) designed for Polytechnic and Diploma students. It provides study materials, paid courses, demo classes, previous year question papers (PYQ), VVI questions, online tests, live classes, notices, downloads, and many more learning resources in one place.

---

🚀 Features

- 📚 Paid Courses
- 🎥 Demo Classes
- 🔴 Live Classes
- ▶️ Recorded Classes
- 📄 Study Material
- 📘 Notes
- 📑 Previous Year Questions (PYQ)
- ⭐ VVI Questions
- 📝 Online Tests
- 📊 Test Results
- 📥 Downloads
- 🔖 Bookmarks
- 📢 Notice Board
- 📋 Syllabus
- 🎓 Polytechnic Entrance
- 👤 Student Dashboard
- 👨‍💼 Admin Dashboard
- 🔐 OTP Login
- 🌙 Dark Mode
- 📱 Progressive Web App (PWA)
- 🔔 Push Notifications
- 📈 Learning Progress

📦 Installation

Clone the repository:

git clone https://github.com/your-username/Polytechnic-Hub.git

Open the project folder:

cd Polytechnic-Hub

Install dependencies:

npm install

Run locally:

npm run start
